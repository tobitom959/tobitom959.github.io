Logo

Logo vložíme do elementu a opatríme ho atribútom id s hodnotou logo.Id má podobný význam ako class, slúži k priradenie štýlu určitému elementu. Rozdiel oproti class je v tom, že element s určitým id (napr. Tu s id logo) môže byť v celom HTML dokumente iba jeden. Určitú class sme mohli priradiť koľkých elementom sme chceli. Id sa využíva teda ku štýlovanie unikátnych prvkov na stránke, ukážkovým príkladom je napr. Logo, ktoré je na každej stránke len jedno. Ešte sa sem vrátime u štýlovanie.
